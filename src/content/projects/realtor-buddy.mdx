## What it is

A field tool for realtors: showings map with GPS navigation, voice-note capture with automatic Whisper transcription, IRS-compliant mileage tracking with offline sync, and a Claude-powered "AI Buddy" that reads the agent's data and suggests next moves.

Multi-tenant security lives at the database, not the application — Supabase RLS policies on every table, with the local Docker Supabase stack mirroring production for development.
