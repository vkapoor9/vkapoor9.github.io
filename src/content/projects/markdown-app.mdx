## What it is

A SwiftUI markdown reader designed for the people who actually live in markdown — long documents, tabular data, code blocks, and elegant typography. Native targets for macOS, iPadOS, and iOS so the same notes feel right on every device.

A small project, but a tidy proof that the AI workflow extends past the web. SwiftUI is a great test bed for whether an LLM can carry serious work into a typed, rapidly-evolving native framework.
